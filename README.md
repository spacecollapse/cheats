## Как инжектить читы

Для начала инжекта нам нужно скачать сам инжектор
Возьмем самый популярный Extreme

Скачать его - https://github.com/spacecollapse/cheats/releases/download/ex/Extreme.Injector.v3.exe

![image](https://user-images.githubusercontent.com/53594431/198863049-c477223d-2537-4508-b27c-a4ad46ab8200.png)

Далее берем длл которую скачали и перетаскиваем в инжектор

![image](https://user-images.githubusercontent.com/53594431/198863083-0dbdd33f-2d43-4f59-82a7-4e88a1f45b10.png)

Всё!теперь она в инжекторе

![image](https://user-images.githubusercontent.com/53594431/198863095-797cb785-265d-42dd-9ad5-656fbbc5dbcb.png)


Теперь нам нужно выбрать процесс игры 

![image](https://user-images.githubusercontent.com/53594431/198863103-14939410-b3c7-4745-adbb-d2cba929d62d.png)


У нас открывается Process List где мы должны выбрать вашу игру и нажать Select

![image](https://user-images.githubusercontent.com/53594431/198863129-bcc72754-fc55-4151-bcac-6a76e4e8b449.png)


Выбераем и далее просто нажимаем Inject

![image](https://user-images.githubusercontent.com/53594431/198863147-52dd717f-c777-4077-8f1e-84123926dd01.png)
